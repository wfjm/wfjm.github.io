--- title: "GitHub mvs38j-langtest created" tags: ["mvs"] ---

Inspired by the example codes collection SYS2.JCLLIB in the MVS 3.8j Tur(n)-key 4- System I wrote a whole array of test and benchmark codes organized in cases which address different programming aspects

  • hewo - The classical 'Hello Word'
  • sine - Line printer plot of sine and cosine
  • soep - Sieve of Eratosthenes prime search (byte)
  • soeq - Sieve of Eratosthenes prime search (bit)
  • towh - Tower of Hanoi solver
  • mcpi - Monte Carlo estimate of pi
and languages
  • Algol 60
  • Assembler
  • C
  • COBOL
  • FORTRAN-4
  • Pascal
  • PL/I
  • Simula
which yield now a total of 105 test and benchmark jobs. All the above is now available on GitHub under wfjm/mvs38j-langtest with plenty of READMEs on all aspects:
  • overview
  • language-case matrix
  • benchmark summary

The jcl of the jobs is generated by the hercjis script due to the 3-dim matrix structure (case,compiler,job-type) of the project.

Enjoy and let me know any comments you might have.

For original posting to Yahoo! Group - turnkey-MVS see thread 10795. Login required. For the benefit of those not having access to Perl the full set of ready-to-submit jcl files was uploaded to turnkey-mvs/files/mvs38j-langtest.zip.
