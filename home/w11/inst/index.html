Download Sources

All sources are contained in the GitHub w11 repository.

Download Pre-Build Bitfiles

Tarballs with ready to use bit files and and all logfiles from the tool chain can be downloaded for releases since w11a_V0.60 from this site. This area is organized in folders for different releases. The tarball file names contain information about release, Xlinix tool, and design 

  http://www.retro11.de/data/oc_w11/bitkits/release/release_tool_design.tgz

Download OS Kits (disk and tape images)

Tarballs with ready to use operating system image can be downloaded from this site. The tarball file names contain information about OS and media type 

  http://www.retro11.de/data/oc_w11/oskits/ostype_mediatype.tgz

See section Systems for more information.

Documentation

An admittedly still rudimentary documentation can be found in the doc directory, specifically on

  • installation and building test benches and systems INSTALL.md
  • running test benches in w11a_tb_guide.md
  • booting operating systems in w11a_os_guide.md
  • known differences, limitations and issues in README_known_issues.md and w11a_known_issues.md

The shell commands in tools/bin have man pages under tools/man/man1 which are also available online.

Releases

A releases is given in the CHANGELOG.md and in section Releases.
