For all without a good private library of original manuals the bitsavers.org document archive of historical DEC manuals comes to the rescue. Another good source of original manuals was the former {% include wbm-link.html date="20090914043922" link="http://www.computer.museum.uq.edu.au/documents.html" text="UQ Museum of IT" %} , they are marked with a (UQ).

But be aware that all manuals are under the copyright of DEC, now held by HP.

General Handbooks

  • PDP-11 Processor Handbook 1979 (11/70 in Chapter 10; p 277 ff)
  • PDP-11 Peripherals Handbook 1976 (DL11 4-128; KW11-L 4-332; LP11 4-353; PC11 4-376; RK11 4-433)
  • PDP-11 Bus Handbook 1979 (UNIBUS and Q-Bus)
  • PDP-11 UNIBUS Design Description (1979)
  • Massbus Specification (1972)

PDP-11/45 Processor (KB11-A, KT11-C, and FP11-B)

The 11/45 is essentially a 11/70 without cache, massbus and 22 bit addressing, but otherwise very similar.

  • KB11-A Processor Maintenance Manual (1976)
  • {% include wbm-link.html date="20091015221033" link="http://www.computer.museum.uq.edu.au/pdf/DEC-11-HGKTCB-D%20PDP-1145%20Memory%20Management%20Reference%20Manual.pdf" text="KT11-C Memory Management Unit Reference Manual" %} (1973) (UQ)
  • KT11-C Memory Management Unit Maintenance Manual (1974)
  • {% include wbm-link.html date="20090915082041" link="http://www.computer.museum.uq.edu.au/pdf/DEC-11-HFPAA-C-D%20FP11%20Floating-point%20Processor%20Maintenance%20Manual.pdf" text="FP11-B Floating Point Unit Maintenance Manual" %} (1974) (UQ)
  • PDP-11/45, 11/50, and 11/55 System Maintenance Manual (1976)
  • PDP-11/45 System Engineering Drawings (1974)

J11 Processor
