For all without a good private library of original manuals, the bitsavers.org document archive of historical DEC manuals comes to the rescue. Another good source of original manuals was the former {% include wbm-link.html date="20090914043922" link="http://www.computer.museum.uq.edu.au/documents.html" text="UQ Museum of IT" %} . They are now archived in the www.computer.museum.uq.edu.au folder tree at bitsavers.org and marked with a (UQ).

But be aware that all manuals are under the copyright of DEC, now held by HP.

PDP-11/45 Processor (KB11-A, KT11-C, and FP11-B)

The 11/45 is essentially an 11/70 without cache, massbus and 22-bit addressing, but otherwise very similar.

  • PDP-11/45, 11/50, and 11/55 System Maintenance Manual (1976)
  • PDP-11/45 System Engineering Drawings (1974)

PDP-11/44 Processor (KD11-Z, KK11-B, FP11-F, KE44-A)

The 11/44 is a low-cost 11/70. 22-bit addressing and cache, but without Massbus. Only UNIBUS peripherals. The last PDP-11 built from descrete components.

  • PDP-11/44 System Technical Manual (1980)
  • PDP-11/44 System User's Guide (1981)
  • PDP-11X44 Field Maintenance Print Set (1979)
  • FP11-F Floating Point Processor Technical Manual (1979)
  • FP11-F Maintenance Print Set (1979)
  • KE44-A Commercial Instruction Set Processor (CISP) User's Guide (1981) [OCR]
  • KE44-A Commercial Instruction Set Processor (CISP) Technical Manual (1981) [OCR]
